# svg-logo-maker

## Description

An application that can take user input and selection of color shape and logo name. The combined information will be used to generate SVG logo and save as a .svg file.

## Installation

jest and inquirer

## Executing program

Running the program:\
node index.js\
<br>
Testing:\
npm test

## Usage

Create SVG logo
